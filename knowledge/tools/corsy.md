## Overview

CORS misconfiguration scanner.

## Command Reference

```bash
python3 /opt/tools/Corsy/corsy.py -u https://{domain} -t 20 -o {domain}_cors_results.txt
echo "[+] Corsy: CORS scan completed"
```

## Features

- CORS testing
- Misconfiguration detection
- Multi-threaded
