## Overview

Free and open-source web application security scanner with automated and manual testing capabilities.

## Command Reference

```bash
zap-baseline.py -t https://{domain} -J {domain}_zap_baseline.json
echo "[+] ZAP: Baseline security scan completed"
```

## Features

- Automated scanning
- Manual testing
- API support
