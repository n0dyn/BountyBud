## XSS Injection Contexts

Understanding the injection context is critical for crafting effective XSS payloads.

### HTML Context

Payload will be inserted into HTML content

### HTML Attribute

Payload will be inserted into an HTML attribute

### Script Context

Payload will be inserted into JavaScript code

### CSS Context

Payload will be inserted into CSS styles

### URL Context

Payload will be part of a URL

### HTML Comment

Payload will be inserted into HTML comments
