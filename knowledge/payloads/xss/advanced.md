- **Contexts**: script
- **Severity**: high


### Event Handler Chain

Chain multiple event handlers for stealth

- **Contexts**: html
- **Severity**: medium

```html
<div onmouseover="this.onclick=function(){alert(1)}" onclick="">Hover then click</div>
```

- **Severity**: high
